GitHub Pages 使用方式：

1. 刪掉原專案中舊的 index.html、state.min.json、asset_manifest_full.json、assets/、models/。
2. 把這個資料夾內所有內容放到 GitHub Pages 發布資料夾根目錄。
3. 必須保留：index.html、state.min.json、asset_manifest_full.json、assets/、models/。
4. 不要再使用 index_lite.html、index_extracted.html、gallery.html、SinTower.glb。
5. model_0004.bin 是 SinTower.glb 改名後的 GLB 二進位檔，單檔低於 GitHub 100MB 限制。
