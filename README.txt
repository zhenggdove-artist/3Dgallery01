Sin Tower Gallery GitHub Pages package

把這個資料夾裡的 index.html、state.min.json、asset_manifest_full.json 複製到你本機的 gallery_extracted_v2 資料夾。
不要刪除 gallery_extracted_v2 裡的 assets/ 與 models/。

最後 GitHub Pages 發布資料夾必須是：

index.html
state.min.json
asset_manifest_full.json
assets/
models/

assets/ 與 models/ 必須使用你本機 v2 腳本拆出的原始資料夾。
